A. 取球游戏

题意

桌子上放着 nnn 个球，第 iii 个球的编号为 iii。取出 mmm 个球，求球的编号的最小值恰好为 kkk 的方案数。结果对 109+710^9 + 7109+7 取模。

数据范围

1≤n,m≤106,1≤k≤n1\le n,m\le10^6,1\le k\le n1≤n,m≤106,1≤k≤n。

思路

不能取编号为 1∼k−11\sim k-11∼k−1 的球；一定取编号为 kkk 的球；剩下 n−kn-kn−k 个球中选出 m−1m-1m−1 个球。

所以答案为 (n−km−1)\dbinom{n-k}{m-1}(m−1n−k​)。

需要求逆元，快速幂模板省略。

代码

#include <iostream>
#define f(x, y, z) for (register int x = (y); (x) <= (z); (x)++)
using namespace std;
typedef long long ll;
const int N = 1e6 + 10;
const ll MOD = 1e9 + 7;

ll ksm(ll a, ll x) { ... }

ll C(int n, int m) {
	if (m > n) return 0;
	if (n == m) return 1;
	if (m == 0) return 1;
	ll a = 1, b = 1;
	f(i, n - m + 1, n) a = a * i % MOD;
	f(i, 2, m) b = b * i % MOD;
	return a * ksm(b, MOD - 2) % MOD;
}

	cin.tie(0), cout.tie(0);
	
	ll n, m, k;
	cin >> n >> m >> k;
	cout << C(n - k, m - 1) << '\n';
	
	return 0;
}

B. 公约数问题

题意

给定正整数 nnn，请你解决下面两个子任务：

  1. 求有多少对 (i,j)(i, j)(i,j) 满足 1≤i,j≤n1\le i,j\le n1≤i,j≤n 且 gcd⁡(i,j)=1\gcd (i,j) = 1gcd(i,j)=1；
  2. 求有多少对 (i,j)(i, j)(i,j) 满足 1≤i,j≤n1\le i ,j\le n1≤i,j≤n 且 gcd⁡(i,j)\gcd(i,j)gcd(i,j) 为素数。

输入第一行为 T∈{0,1}T\in\{0,1\}T∈{0,1}，代表该测试点要完成的子任务编号。

数据范围

对于子任务 1 和 2，均有 1≤n≤1071\le n\le10^71≤n≤107。

思路

子任务 1

由于 (i,j)(i,j)(i,j) 和 (j,i)(j,i)(j,i) 成对出现，所以只需要统计 j≤ij\le ij≤i 的情况，最后乘 222 再减 111（因为有一个 (1,1)(1,1)(1,1) 重复计算）即为答案。

∑i=1n∑j=1n[gcd⁡(i,j)=1]=2×∑i=1n∑j=1i[gcd⁡(i,j)=1]−1=2×∑i−1nφ(i)−1.\begin{aligned} &\sum_{i=1}^n\sum_{j=1}^n[\gcd(i,j)=1]\\ =&2\times\sum_{i=1}^n\sum_{j=1}^i[\gcd(i,j)=1]-1\\ =&2\times\sum_{i-1}^n\varphi(i)-1. \end{aligned} ==​i=1∑n​j=1∑n​[gcd(i,j)=1]2×i=1∑n​j=1∑i​[gcd(i,j)=1]−12×i−1∑n​φ(i)−1.​

发现答案即为欧拉函数的前缀和。在线性筛之后处理前缀和即可。

时间复杂度 O(n)O(n)O(n)。

子任务 2

∑i=1n∑j=1n[gcd⁡(i,j)∈P]=∑p∈P∑i=1n∑j=1n[gcd⁡(i,j)=p]=∑p∈P∑i=1n∑j=1n[gcd⁡(ip,jp)=1]=∑p∈P[2×∑i−1nφ(ip)−1].\begin{aligned} &\sum_{i=1}^n\sum_{j=1}^n[\gcd(i,j)\in\mathbb P]\\ =&\sum_{p\in\mathbb P}\sum_{i=1}^n\sum_{j=1}^n[\gcd(i,j)=p]\\ =&\sum_{p\in\mathbb P}\sum_{i=1}^n\sum_{j=1}^n\left[\gcd\left(\frac ip,\frac jp\right)=1\right]\\ =&\sum_{p\in\mathbb P}\left[2\times\sum_{i-1}^n\varphi\left(\frac ip\right)-1\right]. \end{aligned} ===​i=1∑n​j=1∑n​[gcd(i,j)∈P]p∈P∑​i=1∑n​j=1∑n​[gcd(i,j)=p]p∈P∑​i=1∑n​j=1∑n​[gcd(pi​,pj​)=1]p∈P∑​[2×i−1∑n​φ(pi​)−1].​

其中 P\mathbb PP 表示全体质数构成的集合。

线性筛处理出 nnn 以内的质数后，枚举所有的质数，像子任务 1 一样求出答案并累加起来，即为所求。

代码

#pragma GCC optimize(2)
#include <iostream>
#include <bitset>
#include <cmath>
#define f(x, y, z) for (register int x = (y); (x) <= (z); (x)++)
using namespace std;
typedef long long ll;
const int N = 1e7;
int T, n;
ll ans;
bitset<N + 10> ck;
int pri[700000], cnt;
int phi[N];
ll sum[N]; //欧拉函数前缀和

inline void get_phi() {
	f(i, 2, n) {
		if (!ck[i]) pri[++cnt] = i, phi[i] = i - 1;
		f(j, 1, cnt) {
			int k = i * pri[j];
			if (k > N) break;
			ck[k] = true;
			if (!(i % pri[j])) {
				phi[k] = phi[i] * pri[j];
				break;
			}
			phi[k] = phi[i] * (pri[j] - 1);
		}
	}
	f(i, 2, n) sum[i] = sum[i - 1] + phi[i];
}

int main() {
	
	cin >> T >> n;
	get_phi();
	if (T == 1) {
		cout << (sum[n] << 1 | 1) << '\n';
	} else if (T == 2) {
		f(i, 1, cnt) ans += (sum[n / pri[i]] << 1 | 1);
		cout << ans << '\n';
	}
	
	return 0;
}

思路与代码

子任务 1（扩展欧几里得算法）

exgcd 模板。

ax≡1(modp)  ⟺  xa+yp=1ax\equiv1\pmod p\iff xa+yp=1ax≡1(modp)⟺xa+yp=1。用 exgcd 求出 x,yx,yx,y 即可。

题中要求 xxx 为最小正整数解，则 x←(x+p) mod px\leftarrow(x+p)\bmod px←(x+p)modp。

namespace task1 {
	int exgcd(int a, int b, int &x, int &y) {
		if (!b) {
			x = 1, y = 0;
			return a;
		}
		int g = exgcd(b, a % b, x, y);
		int t = x;
		x = y;
		y = t - a / b * y;
		return g;
	}
	void solve1() {
		while (T--) {
			int x, y, a, b;
			cin >> a >> b;
			exgcd(a, b, x, y);
			cout << (x + b) % b << '\n';
		}
		return;
	}
}

时间复杂度 O(Tlog⁡p)O(T\log p)O(Tlogp)。

子任务 2（费马小定理，试除求阶）

注意到问题的形式与费马小定理 ap−1≡1(modp)a^{p-1}\equiv1\pmod pap−1≡1(modp) 颇为相似。

可不可以由 x0=p−1x_0=p-1x0​=p−1 求出最小整数解呢？

猜想：若 am≡1(modp)a^m\equiv1\pmod pam≡1(modp)，且 ax≡1(modp)a^x\equiv1\pmod pax≡1(modp) 的最小正整数解是 xxx，那么 x∣mx\mid mx∣m。

证明：假设 x∤mx\nmid mx∤m，那么可设 m=kx+bm=kx+bm=kx+b，其中 0<b<x0<b<x0<b<x。

于是 akx+b≡1(modp)a^{kx+b}\equiv1\pmod pakx+b≡1(modp)，所以 ab≡1(modp)a^b\equiv1\pmod pab≡1(modp)，即存在 0<b<x0<b<x0<b<x 使得方程成立，与 xxx 为最小正整数解矛盾。因此假设不成立，原命题得证。

所以，我们只需要令 x=p−1x=p-1x=p−1，然后不断用 xxx 去除以它的最小质因子，直到 ax≡1(modp)a^x\not\equiv1\pmod pax​≡1(modp)，此时上一个 xxx 即为最小正整数解。

子任务 3（推式子 + 数论分块）

引理：n=∑d∣nφ(d)n=\sum\limits_{d\mid n}\varphi(d)n=d∣n∑​φ(d)。

证明：将正整数 1,2,…,n1,2,\dots,n1,2,…,n 按其和 nnn 的最大公约数分成若干集合。这些集合两两互斥，且并集为 {1,2,…,n}\{1,2,\dots,n\}{1,2,…,n}。

若用 τ(n)\tau(n)τ(n) 表示 nnn 的约数个数，那么这些集合一共有 τ(n)\tau(n)τ(n) 个，且每一个 d∣nd\mid nd∣n 与分出的一个集合一一对应。

显然，这些集合两两互斥，且并集为 {1,2,…,n}\{1,2,\dots,n\}{1,2,…,n}。所以，有 n=∑d∣n∑i=1n[gcd⁡(i,n)=d]n=\sum\limits_{d\mid n}\sum\limits_{i=1}^n[\gcd(i,n)=d]n=d∣n∑​i=1∑n​[gcd(i,n)=d]。

由 gcd⁡\gcdgcd 的性质，上式等于 ∑d∣n∑i=1n[gcd⁡(id,nd)=1]\sum\limits_{d\mid n}\sum\limits_{i=1}^{n}\left[\gcd\left(\dfrac id,\dfrac nd\right)=1\right]d∣n∑​i=1∑n​[gcd(di​,dn​)=1]。

由欧拉函数的定义，上式等于 ∑d∣nφ(nd)\sum\limits_{d\mid n}\varphi\left(\dfrac nd\right)d∣n∑​φ(dn​)。

由于每一个 ddd 都对应一个 nd\dfrac nddn​，上式即为 ∑d∣nφ(d)\sum\limits_{d\mid n}\varphi(d)d∣n∑​φ(d)。证毕。

接下来大力推式子：

∑i=1n∑j=1mgcd⁡(i,j)=∑i=1n∑j=1m∑d∣gcd⁡(i,j)φ(d)=∑i=1n∑j=1m∑d∣i∧d∣jφ(d)=∑d=1nφ(d)∑i=1n∑j=1m∑d∣i∧d∣j1=∑d=1nφ(d)(∑i=1n[d∣i])(∑j=1m[d∣j])=∑d=1nφ(d)⌊nd⌋⌊md⌋.\begin{aligned} & \sum_{i=1}^n\sum_{j=1}^m\gcd(i,j)\\ =& \sum_{i=1}^n\sum_{j=1}^m\sum_{d\mid\gcd(i,j)}\varphi(d)\\ =& \sum_{i=1}^n\sum_{j=1}^m\sum_{d\mid i\wedge d\mid j}\varphi(d)\\ =& \sum_{d=1}^n\varphi(d)\sum_{i=1}^n\sum_{j=1}^m\sum_{d\mid i\wedge d\mid j}1\\ =& \sum_{d=1}^n\varphi(d)\left(\sum_{i=1}^n[d\mid i]\right)\left(\sum_{j=1}^m[d\mid j]\right)\\ =& \sum_{d=1}^n\varphi(d)\left\lfloor\frac nd\right\rfloor\left\lfloor\frac md\right\rfloor. \end{aligned} =====​i=1∑n​j=1∑m​gcd(i,j)i=1∑n​j=1∑m​d∣gcd(i,j)∑​φ(d)i=1∑n​j=1∑m​d∣i∧d∣j∑​φ(d)d=1∑n​φ(d)i=1∑n​j=1∑m​d∣i∧d∣j∑​1d=1∑n​φ(d)(i=1∑n​[d∣i])(j=1∑m​[d∣j])d=1∑n​φ(d)⌊dn​⌋⌊dm​⌋.​

先处理出欧拉函数的前缀和，然后用数论分块：算出 ⌊nd⌋\left\lfloor\dfrac nd\right\rfloor⌊dn​⌋ 和 ⌊md⌋\left\lfloor\dfrac md\right\rfloor⌊dm​⌋ 均相等的每一段 ddd 的长度，乘以 φ(d)\varphi(d)φ(d) 的值并累加。

namespace task3 {
	const int N = 1e7 + 10; int pri[700000], cnt, phi[N]; bitset<N> ck;
	void get_phi() { ... }
	void solve3() {
		get_phi();
		while (T--) {
			int n, m, ans = 0;
			cin >> n >> m;
			if (n > m) swap(n, m);
			int l = 1, rn, rm, r;
			while (l <= n) {
				rn = n / (n / l), rm = m / (m / l);
				r = min(rn, rm);
				ans += (phi[r] - phi[l - 1]) * (n / l) * (m / l);
				l = r + 1;
			}
			cout << ans << '\n';
		}
	}
}

子任务 4（变换 -> 线性筛约数个数）

结论：令 τ(i)\tau(i)τ(i) 表示 iii 的正因数个数，那么 ∑i=1n⌊ni⌋=∑i=1nτ(i)\sum\limits_{i=1}^n\left\lfloor\dfrac ni\right\rfloor=\sum\limits_{i=1}^n\tau(i)i=1∑n​⌊in​⌋=i=1∑n​τ(i)。

感性理解：

∑i=113⌊13i⌋=∑j=113τ(j)=37\sum\limits_{i=1}^{13}\left\lfloor\dfrac {13}i\right\rfloor=\sum\limits_{j=1}^{13}\tau(j)=37i=1∑13​⌊i13​⌋=j=1∑13​τ(j)=37。

于是可以线性预处理出 τ\tauτ 的前缀和 sss，输出 s(n)s(n)s(n) 即为答案。

namespace task4 {
	const int N = 1e7 + 10;
	int pri[700000], cnt, t[N], g[N];
	bitset<N> ck;
	void get_tau() {
		t[1] = 1;
		f(i, 2, 1e7) {
			if (!ck[i]) pri[++cnt] = i, g[i] = t[i] = 2;
			f(j, 1, cnt) {
				int k = i * pri[j];
				if (k > 1e7) break;
				ck[k] = true;
				if (!(i % pri[j])) {
					g[k] = g[i] + 1;
					t[k] = t[i] / g[i] * (g[i] + 1);
					break;
				}
				g[k] = 2;
				t[k] = t[i] * 2;
			}
		}
		f(i, 2, 1e7) t[i] += t[i - 1];
	}
	void solve4() {
		get_tau();

			int n;

}
