Luogu P2606 [ZJOI2010]排列计数 题解

题目描述

称一个 1∼n1 \sim n1∼n 的排列 p1,p2,…,pnp_1,p_2, \dots ,p_np1​,p2​,…,pn​ 是 Magic 的，当且仅当

∀i∈[2,n],pi>p⌊i/2⌋.\forall i \in [2,n],p_i > p_{\lfloor i/2 \rfloor}. ∀i∈[2,n],pi​>p⌊i/2⌋​.

计算 1∼n1 \sim n1∼n 的排列中有多少是 Magic 的，答案可能很大，只需输出模 mmm 以后的值。

思路

把题目中给的条件转化一下，变成：对于每一个 iii，满足 pi<p2ip_i<p_{2i}pi​<p2i​ 且 pi<p2i+1p_i<p_{2i+1}pi​<p2i+1​。

发现 2i2i2i 和 2i+12i+12i+1 即为二叉树上节点 iii 的两个儿子，上述性质符合小根堆的性质。

设 sis_isi​ 表示以 iii 为根的子树大小，fif_ifi​ 表示用 1,2,…,si1,2,\dots,s_i1,2,…,si​ 填满以 iii 为根的子树且满足上述性质的方案数。显然，用 1,2,…,si1,2,\dots,s_i1,2,…,si​ 和 2,3,…,si+12,3,\dots,s_i+12,3,…,si​+1 填的方案是等效的，即方案数与 iii 本身的值无关，而与 sis_isi​ 的大小有关。

考虑在二叉树上 DP：从叶子到根，先由子树的大小算出 sis_isi​，然后再由子树的方案数和 sis_isi​算出 fif_ifi​。设两个子树分别是 lll 和 rrr，则需要从 sis_isi​ 个数中选出 sls_{l}sl​ 和 srs_{r}sr​ 个分别给两个子树，方案数为 (si−1sl)\dbinom{s_i-1}{s_{l}}(sl​si​−1​)。于是，根据分步乘法原理，转移方程为

fi=fl×fr×(si−1sl).f_i=f_{l}\times f_{r}\times\dbinom{s_i-1}{s_{l}}. fi​=fl​×fr​×(sl​si​−1​).

最后的答案为 f1f_1f1​。

代码

  • 由于叶子节点没有子树，fff 数组的初值须全部设为 111，不然会输出 0；
  • 并且，sss 数组和 fff 数组需要开大一倍，不然会 RE；
  • 由于模数很小，可以用 Lucas 定理求组合数。
  • （预处理出阶乘比直接算要快 0.4 秒左右，不过都能过）
  • （fill 函数是 stl 库中用于给数组内所有元素赋值的函数，与 memset 不同）
  • （十年 OI 一场空，不开 long long 见祖宗）

#include <iostream>
#include <algorithm>
#define f(x, y, z) for (int x = (y); (x) <= (z); ++(x))
#define g(x, y, z) for (int x = (y); (x) >= (z); --(x))
#define il inline
#define ls (i << 1)
#define rs (i << 1 | 1)
#define int ll
using namespace std;
typedef long long ll;
const int N = 1e6 + 10;
int n, p, s[N << 1], f[N << 1];
int fac[N];

il void Init(int p) {
	fac[0] = fac[1] = 1;
	f(i, 2, n) fac[i] = fac[i - 1] * i % p;
}

il int ksm(int a, int x, int p) { ... }

il int C(int n, int m, int p) {
//	int a = 1, b = 1;
//	f(i, n - m + 1, n) a = a * i % p;
//	f(i, 2, m) b = b * i % p;
//	return a * ksm(b, p - 2, p) % p;
	return fac[n] * ksm(fac[m] * fac[n - m] % p, p - 2, p) % p;
}

int Lucas(int n, int m, int p) {
	if (!m) return 1;
	return C(n % p, m % p, p) * Lucas(n / p, m / p, p) % p;
}

signed main() {
	
	cin >> n >> p;
	Init(p);
	fill(f, f + (N << 1), 1);
	g(i, n, 1) {
		s[i] = s[ls] + s[rs] + 1;
		f[i] = f[ls] * f[rs] % p * Lucas(s[i] - 1, s[ls], p) % p;
	}
	cout << f[1] << '\n';
	
	return 0;
}
